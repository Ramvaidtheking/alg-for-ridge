Fire Weather Index (FWI) Prediction

The Fire Weather Index (FWI) is a numerical rating of fire intensity, used to estimate wildfire risk based on weather conditions. It is widely used in forestry, disaster management, and environmental monitoring.

Components of the FWI System

  • Fine Fuel Moisture Code (FFMC) : Measures surface litter dryness.
  • Duff Moisture Code (DMC) : Indicates moisture in decomposed organic layers.
  • Drought Code (DC) : Reflects moisture in deep soil layers.
  • Initial Spread Index (ISI) : Predicts fire spread based on wind & dryness.
  • Build-Up Index (BUI) : Represents the total fuel available.
  • Fire Weather Index (FWI) : Measures fire intensity and spread potential.

How is FWI Used?

  • Predicts wildfire danger levels for fire prevention.
  • Used in forest management and controlled burns.
  • Supports firefighting efforts and resource allocation.
  • Helps in climate monitoring and wildfire trend analysis.

Predicting FWI with Algerian Forest Fire Dataset

The dataset includes 244 instances from two regions in Algeria: Bejaia (0) and Sidi Bel-abbes (1), covering June-September 2012.

Input Features Required:

  1. Temperature (°C)
  2. Relative Humidity (%)
  3. Wind Speed (km/h)
  4. Rainfall (mm)
  5. Fine Fuel Moisture Code (FFMC)
  6. Duff Moisture Code (DMC)
  7. Initial Spread Index (ISI)
  8. Classes (0: No Fire, 1: Fire)
  9. Region (0: Bejaia, 1: Sidi Bel-abbes)

How to Use:

  1. Enter temperature, humidity, wind speed, and other values.
  2. Click Predict, and the model will calculate the FWI index.
  3. Higher FWI indicates higher wildfire risk.
